Midipipe: stdin/stdout <-> alsa seq midi device
---

```
Usage: midipipe [-l] [-h] [-i|-o] [-b] [device]
  -l: List devices (can be combined with -i|-o)
  -i: Input from device only
  -o: Output to device only
  -b: Binary in-/output (instead of Hex)
  -h: Show Usage
```
